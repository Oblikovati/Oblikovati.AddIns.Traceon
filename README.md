# Oblikovati Traceon add-in

Electron-optics simulation for Oblikovati: a radially-symmetric **Boundary Element
Method (BEM)** electrostatic/magnetostatic field solver and charged-particle tracer.
It sections a body into an axisymmetric profile, solves the surface charge
distribution, traces rays through the resulting field, and renders fields and
trajectories back into the viewport as client graphics.

This is a **pure-Go** port of the open core of the
[Traceon](https://github.com/leon-vv/Traceon) library (MPL-2.0) — no cgo solver, no
subprocess. The numerical core (`core/`) is a standalone, oracle-verified translation
with **no host dependency**; the engine (`engine/`) bridges it to the host over the
Apache-2.0 `oblikovati.org/api`. Built and shipped like the FEMM/CalculiX bridges: a
cgo `c-shared` shell (`export.go`, `hostcaller.go`) over the C ABI.

Scope: the open Traceon core is **radially symmetric only**. Full 3D (solver, tracer,
fast-multipole, Coulomb interaction) is the closed-source `traceon_pro` and is out of
scope.

## Layout

```
export.go, hostcaller.go, manifest.*   cgo c-shared shell (the only cgo)
core/      pure-Go numerical port (cgo-free, NO api dependency) — the equivalence deliverable
engine/    host bridge: geometry → core → client-graphics
cmd/       standalone CLI harnesses
tools/     gen_fixtures.py — golden-fixture generator (oracle)
```

## Build & test

```
make build          # c-shared library into build/ (CGO_ENABLED=1)
make install        # build + copy into the host's addins/ dir
make test           # pure-Go core + engine tests (CGO_ENABLED=0)
make verify-oracle  # regenerate golden fixtures from upstream Traceon, re-run core tests
```

## Oracle / numerical equivalence

Each `core/<pkg>` is verified against the upstream Traceon as oracle, two ways:
golden-fixture JSON generated by `tools/gen_fixtures.py` (no Python at test time) and
ported analytic-oracle assertions (scipy.quad / Biot–Savart closed forms). Tolerances
mirror `np.isclose` (atol 1e-8, rtol 1e-5), tightened per module.
